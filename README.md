# Music Bot for Discord

## How to use
    
    export DISCORD_BOT_TOKEN=
    docker build . -t discordmusic && docker run -it --rm -e DISCORD_BOT_TOKEN=$DISCORD_BOT_TOKEN --name=discordmusic discordmusic
